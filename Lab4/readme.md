# Lab 4

### LDHT_Sensor
這是溫濕度感測器的範例

Hint: 記得要先呼叫 `dht.read()` 才會將資料讀進來！

之後 `dht.readTemperature()` 才會有值

### ADXL345_demo_code
這邊是三軸加速器的範例

他會讀x, y, z 三軸的加速度

請去判斷三軸加速器時會有什麼變化

### WifiWebClient
這邊是Wifi連線的範例

會先看到 `Connecting to AP`，表示他正在連 AP

如果成功連上 AP 的話，會看到 `Connecting to WebSite`，表示說他開始去連網站

之後就可以在 Serial Monitor 上看到跟 `google.com` 正在傳訊息
